Quick Start

Get up and running in minutes with our simple setup process

Terminal bash 
# Clone the repository
git clone https://github.com/xcrafter/zatca-api.git
cd zatca-api

# Run setup script
./setup-deployment.sh

# Start the application
docker-compose up -d

# Access the platform
# Web UI: localhost:3000/ui
# Docs: localhost:3000/api-docs
