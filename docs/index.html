Quick Start

# Clone the repository
git clone https://github.com/xcrafter/zatca-api.git
cd zatca-api

# Run setup script
./setup-deployment.sh

# Start the application
docker-compose up -d

# Access the platform
# Web UI: localhost:3000/ui
# Docs: localhost:3000/api-docs

Requires Docker and a PostgreSQL database. See the full documentation for detailed setup instructions.
